# 价值观优先级评估数据集与代码库

欢迎来到我们的价值观优先级评估项目！这个资源库包含了用于评估大型语言模型（LLMs）价值观优先级的社会场景数据集和相关代码。这些资源是为了促进对AI基于价值观进行决策的理解。

## 数据集

我们的**社会场景决策数据集**包含1613个社会场景和3226个基于价值观的决策的数据集。每个场景都涉及两个相互冲突的决策，关联相应的价值观。
